# Usage


## 1.Clone

```bash
git clone https://github.com/piroyoung/python_null_project

```
## 2.Rename
Please rewrite name of package `pir` as you need. name of packages are included in below.

- `setup.py`
- directory name `./pir`


## 3.Test
```bash
python3 setup.py test
```

## 4.Install
```bash
python3 setup.py install
```

## 5.Use
```python
import pir
```

## 6.Uninstall
```bash 
pip3 uninstall pir

```
